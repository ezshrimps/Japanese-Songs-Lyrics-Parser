# Japanese-Songs-Lyrics-Parser
The idea is extracted from Bilibili account 卷心儿. The core functioni of the Lyrics Parser is to receive a japanese song lyrics input from the user, and mark the hiragana of its kanji, romaji of the whole sentence, Chinese translation(or any chosen language), and seperate them to difference grammar.
